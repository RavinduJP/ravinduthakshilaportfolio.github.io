library Management System (mobile application)

This is an android mobile application built using native android technologies such as SQLite, basic CRUD operations, and user login systems. Basically, this application can maintain all the library functions. There are book maintenance functions (add, update, remove books), admin login, user login, and members login in this app. This is lite weight mobile application and SQLite as the backend part and also the android material design as the front end part of this app.
