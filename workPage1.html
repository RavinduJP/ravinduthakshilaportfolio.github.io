Employee Management System (Web-based application)

This is an Employee Management System web-based application built using the Spring Boot framework and React js JavaScript library. There are CRUD operations, (Create, Read, Update, Delete functions), a database facility, creative UI, and simple handling that contain this application. if Consider the backend part, there are several APIs that handle the data passing front end to back end.
